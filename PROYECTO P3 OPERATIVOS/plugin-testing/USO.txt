TESTING PLUGIN - EJECUTABLE LINUX
==================================

EJECUTABLE:
  /home/jordan/Escritorio/PROYECTO\ P3\ OPERATIVOS/plugin-testing/dist/testing-plugin

COMO USAR:
  1. Ejecuta el archivo binario:
     ./dist/testing-plugin

  2. El plugin automáticamente:
     - Ejecuta todas las pruebas
     - Abre el navegador en http://localhost:3002
     - Guarda los reportes en la BD

DATOS:
  - Los reportes se almacenan en: ~/.testing-plugin/db/reports.json
  - Los reportes PDF se guardan en: ~/.testing-plugin/reports/

PUERTO:
  - Servidor corriendo en: http://localhost:3002

CARACTERISTICAS:
  ✓ Ejecución automática de tests (FUNCTIONAL, NON-FUNCTIONAL, LOAD, STRESS)
  ✓ Interfaz visual moderna en React
  ✓ Base de datos JSON para almacenar resultados
  ✓ Descarga de reportes en PDF
  ✓ Respuesta de API REST en http://localhost:3002/api/latest-report

REQUERIMIENTOS:
  - Linux x64
  - Puerto 3002 disponible
